# Treinando CNN com Imagens – Classificação Homer vs Bart

Este projeto tem como objetivo desenvolver um modelo de Rede Neural Convolucional (CNN) para realizar a classificação de imagens entre dois personagens da série The Simpsons: Homer Simpson e Bart Simpson.

O problema abordado consiste em uma classificação binária de imagens, na qual o modelo é treinado para identificar padrões visuais presentes nas imagens e, a partir disso, determinar a qual personagem a imagem pertence.
Para isso, são utilizadas técnicas de Deep Learning, permitindo que a rede neural aprenda automaticamente características relevantes das imagens, como formas, cores e traços distintivos dos personagens.

Ao final do treinamento, o modelo é capaz de analisar novas imagens e classificá-las corretamente como Homer ou Bart, demonstrando a aplicação prática de CNNs em tarefas de reconhecimento visual.
