# Treinando-CNN-Com-Imagens-RN
 objetivo deste projeto é desenvolver um modelo de Rede Neural Convolucional (CNN) capaz de classificar imagens entre dois personagens: Homer e Bart, da série Os Simpsons. Aonde se trata de um problema de classificação de imagens binária, onde a rede aprende padrões visuais para distinguir os personagens.
